Questão 1

Implemente um programa em C++ que trabalhe com manipulação de arquivos. A função principal deve:

  1. Ler dados de alunos (nome e nota) do arquivo "alunos.txt"
  2. Calcular a média das notas
  3. Criar um novo arquivo "aprovados.txt" com os alunos acima da média
Exemplo: Entrada: "João 8.5, Maria 7.0, Pedro 9.0"
Saída: "João 8.5, Pedro 9.0" (aprovados.txt)

Questão 2

Implemente uma estrutura Data e uma função compararDatas que compare duas datas. Retorne: 1 (primeira mais recente), -1 (segunda mais recente), 0 (iguais). Inclua validação.

Exemplo: Data1(15,3,2024), Data2(20,2,2024) → Saída: 1

Questão 3

Desenvolva uma calculadora com ponteiros para funções (soma, subtração, multiplicação, divisão). Valide entradas e formate saídas.

Exemplo: "5.2 + 3.8 = 9.0"

Questão 4

Implemente filtros encadeados usando array de ponteiros para funções:

  1. Limitar valores (0-100)
  2. Ajustar para número par
  3. Exibir valor após cada filtro
Exemplo: 123 → 100 → 100 (após filtros)

Questão 5

Desenvolva uma agenda de contatos usando um array de estruturas com as seguintes funcionalidades:

  1. Criar uma estrutura Contato com nome e telefone
  2. Implementar função para adicionar contato
  3. Implementar função para listar todos os contatos
  4. Implementar função para buscar contato por nome
Exemplo:
Adicionar("João", "21999887766");
Buscar("João"); // Exibe: Nome: João, Tel: 21999887766
Dicas:
  1. Defina um tamanho máximo para o array (ex: 100)
  2. Use uma variável para controlar quantos contatos já foram adicionados
  3. Verifique se há espaço antes de adicionar
